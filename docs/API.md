# API Reference

Base URL: `http://localhost:8000/api/v1`

Interactive docs are auto-generated by FastAPI:
- Swagger UI: `/docs`
- ReDoc: `/redoc`
- OpenAPI schema: `/api/v1/openapi.json`

All protected endpoints require an `Authorization: Bearer <access_token>` header.

## Authentication

### POST `/auth/register`
Create a new account.

```json
{ "email": "user@example.com", "password": "password123", "full_name": "Jane" }
```
Returns `201` with the created user (password never returned).

### POST `/auth/login`
Form-encoded (`application/x-www-form-urlencoded`) with `username` and `password`.
Returns:
```json
{ "access_token": "...", "refresh_token": "...", "token_type": "bearer" }
```

### POST `/auth/refresh`
```json
{ "refresh_token": "..." }
```
Returns a fresh token pair.

### GET `/auth/me`
Returns the authenticated user.

## Datasets

### POST `/datasets/upload`
`multipart/form-data` with fields: `file` (required), `name`, `description`.
Accepts `.csv`, `.xlsx`, `.xls`, `.json`. Max size configurable via `MAX_UPLOAD_MB`.

### POST `/datasets/connect-sql`
```json
{
  "name": "Q1 sales",
  "connection_uri": "postgresql+psycopg2://user:pass@host:5432/db",
  "query": "SELECT * FROM sales LIMIT 1000"
}
```
Only read-only `SELECT`/`WITH` queries are permitted.

### GET `/datasets`
List datasets owned by the current user.

### GET `/datasets/{dataset_id}`
Fetch a single dataset.

### DELETE `/datasets/{dataset_id}`
Delete a dataset and its stored file. Returns `204`.

## Analysis

### POST `/analysis/{dataset_id}/run`
Optional query param `target` selects the ML target column. Returns the full
analysis result: cleaning actions, quality report, EDA, feature importance,
insights, KPIs, and executive summary.

### POST `/analysis/query`
```json
{ "dataset_id": "...", "question": "What is the average revenue?" }
```
Returns an answer plus (when an LLM key is configured) the generated code and a
result preview. Falls back to a deterministic engine without a key.

### POST `/analysis/{dataset_id}/summary`
Returns an executive summary for the dataset.

## Reports

### POST `/reports/generate`
```json
{ "dataset_id": "...", "fmt": "pdf", "title": "Optional title" }
```
`fmt` is `pdf` or `excel`. Returns the binary file as an attachment.

## Errors

| Status | Meaning |
| --- | --- |
| 400 | Bad request / invalid SQL |
| 401 | Missing or invalid token |
| 404 | Resource not found or not owned by user |
| 413 | Upload exceeds size limit |
| 415 | Unsupported file type |
| 422 | Validation error (details array) |
| 500 | Internal server error (masked) |
